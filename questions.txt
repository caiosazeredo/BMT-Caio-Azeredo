Qual é a principal vantagem da metodologia agil para o desenvolvimento de software?
Uma empresa de software precisa de um novo aplicativo para atender a um novo mercado. O departamento de engenharia de software começa a trabalhar no projeto, mas percebe que não tem a expertise necessária para desenvolver o aplicativo.

Que tipo de parceria a empresa de software deve procurar para desenvolver o aplicativo?

A empresa de software deve procurar uma parceria com uma empresa de desenvolvimento de software.
